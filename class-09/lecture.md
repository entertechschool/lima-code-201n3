## Estructura de un formulario en HTML

### Tipos de la etiqueta input:

- text
- email
- password
- number
- checkbox
- radio
- submit

```html
<input type="text" />
<input type="email" />
<input type="password" />
<input type="number" />
<input type="checkbox" />
<input type="radio" />
<input type="submit" />
```

- Si queremos una opción mas custom para nuestro boton, recordemos que la etiqueta `button` tiene la opción de ser `type='submit`
