## Listas

¿Qué es una lista en `HTML` ?

- Son etiquetas para dar un oden predeterminado a una cantidad elementos
- Objetos que necesitan ser listados
- Son 2 tipo de lista:
  - Ordenas
  - No ordenadas

¿Cómo se crea una lista ordenada?

```html
<!-- ol: ordered list -->
<ol>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ol>
```

1. Elemento 1
2. Elemento 2
3. Elemento 3

¿Cómo se crea una lista desordenada?

```html
<ul>
  <li>Element1</li>
  <li>Element2</li>
  <li>Element4</li>
</ul>
```
